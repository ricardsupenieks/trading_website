# Investify - Stock trading website

## Main features
* User registration
* Current stock prices and price changes
* Buy, sell and short stocks
* Transfering stocks between users
* Transaction history

## Requirements
* PHP version: 7.4 
* MySQL version: 8.0.31-0ubuntu0.22.04.1 
* Composer version ^2.4.4
